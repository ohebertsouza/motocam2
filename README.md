
# MotoCam2

Sistema avançado de fotografia computacional para Motorola Edge 30 Neo.

## Objetivo
Entregar uma experiência fotográfica semelhante à de smartphones flagship,
focando em:
- Color science premium
- HDR multi-frame
- White balance inteligente
- Nitidez contextual
- Estilos fotográficos
- Retrato com profundidade avançada

## Stack
- Flutter
- Kotlin
- CameraX
- TensorFlow Lite
- OpenCV

## Roadmap

### V1
- Captura rápida
- Estilos fotográficos
- Sharpen inteligente
- HDR básico
- White balance calibrado

### V2
- HDR multi-frame
- Night Mode
- Portrait depth

### V3
- IA neural
- Vídeo computacional
- Super Resolution
